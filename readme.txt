This plugin will show pakistan standard time

== Description ==

The purpose of this plugin is to provide facility to show pk time in post without knowing the difference

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/firstplugin` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. use short code in post [pk-time]
